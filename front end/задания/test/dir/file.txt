Основные настройки файла package.json:

   Для более удобного управления конфигурацией и пакетами приложения в npm применяется файл конфигурации package.json
   Файл package.json поддерживает возможность описания команд (скриптов), запускать которые можно с помощью такой конструкции:

                       npm <task-name>

    В package.json, там появились наши пакеты, которые мы писали.
